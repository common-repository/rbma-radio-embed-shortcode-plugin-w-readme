RBMA Radio Embed Shortcode allows you to embed Red Bull Music Academy Radio Shows into your Pages and Articles.
Simply enclose the show URL between [rbmaradio]-Shrotcode Tags to embed the iFrame.

== Installation ==

1. Upload `rbmar-shortcode.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
